# Envelopes and Amplitude Shaping

> **Stub Notice:** This page is a conceptual stub to be expanded in a future update.

## Overview

Amplitude shaping over time—commonly understood in synthesis via the Attack, Decay, Sustain, Release (ADSR) envelope—is a phenomenon that scales fractally across the Metric DuPeriod.

This reference page maps how the same amplitude-shaping mechanism manifests at different timescales:
- **Macro-scale:** A structural crescendo over 16 bars
- **Meso-scale:** A wind player's breath articulation or a string player's bow stroke
- **Micro-scale:** The initial transient "click" or "chiff" of a note's attack
