# Musical Topics

The goal of this homepage is to serve as a comprehensive guide to musical analysis and conceptualisation. The structure focuses on core **musical concepts** (pitch, rhythm, timbre, perception) and uses the Prime Period Theory (PPT) framework as a powerful, unified lens to explain them.
